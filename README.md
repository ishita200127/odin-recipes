# Odin Recipes

This project is part of *The Odin Project* foundations course. Its purpose is to create a simple multi-page recipe website using only HTML. The project focuses on structuring content, linking pages, and organizing files in a clean and readable way.

## What This Project Is About

The website includes:

- An `index.html` homepage listing links to several recipes  
- Individual recipe pages containing:
  - A title  
  - An ingredients list  
  - Step-by-step instructions  
  - A link back to the homepage  

## Skills Demonstrated

By completing this project, I practiced:

- Writing valid, semantic HTML  
- Using headings, ordered/unordered lists, and anchor tags  
- Creating internal links using relative file paths  
- Organizing project folders and files  
- Building and testing a simple multi-page website  

## Reflection

I practiced creating links between pages using relative paths, organizing files into folders, and using headings and lists in a more semantic way. I also gained confidence in understanding how a browser reads and renders simple HTML documents.
